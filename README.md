# Нумерологический калькулятор (веб + Telegram-бот)

## Возможности
* Нумерологические расчёты по дате рождения (Life Path) и полному имени (Expression / Destiny)
* Валидированный ввод и обработка ошибок
* История 10 последних расчётов с цветовой маркировкой
* Смена фонового изображения в веб-версии
* Telegram-бот с поддержкой:
  * команд `/life` и `/name`
  * голосовых сообщений (распознавание речи)
  * персональной истории расчётов
* Персонализированные рекомендации и краткая справка о значениях чисел

## Быстрый старт (локальный компьютер)

```bash
# 1. Клонировать репозиторий и перейти в каталог
$ git clone <repo-url> numerology_app && cd numerology_app

# 2. Создать виртуальное окружение
$ python3 -m venv venv && source venv/bin/activate

# 3. Установить зависимости
$ pip install -r requirements.txt

# 4. Запустить веб-приложение
$ python webapp/app.py
# Откройте http://127.0.0.1:5000 в браузере
```

## Запуск Telegram-бота
1. Создайте бота у `@BotFather` и скопируйте токен.
2. Установите переменную окружения:
   ```bash
   export TELEGRAM_BOT_TOKEN="123456:ABC..."
   ```
3. Запустите скрипт:
   ```bash
   python bot/bot.py
   ```
4. Откройте диалог с ботом в Telegram и введите `/start`.

### Распознавание голоса
Для конвертации аудио требуется `ffmpeg`:
```bash
sudo apt install ffmpeg  # для Debian/Ubuntu
```

## Установка на мобильный телефон
### Веб-приложение как PWA
1. Запустите веб-сервер на своём ПК или VPS (см. выше). 
2. Откройте сайт в мобильном браузере (Chrome, Safari). 
3. В меню браузера выберите «Добавить на главный экран» – приложение установится автономно.

### Telegram
* Достаточно иметь установленный клиент Telegram. Бот работает кросс-платформенно (iOS/Android). 
* Для офлайн-использования history хранится в памяти устройства, однако распознавание речи и команды требуют соединения с интернет.

## Деплой на VPS (Ubuntu 22.04)
```bash
# Создать пользователя, установить Python 3.11+
# ...
apt update && apt install python3-venv ffmpeg -y

# Скопировать проект и установить зависимости
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt

# (Опционально) настроить Gunicorn + Nginx для веб-части
# Файл systemd service: /etc/systemd/system/numerology.service
# ...
```


## Добавление фоновых изображений
* Положите файлы `bg1.jpg`, `bg2.jpg`, ... в каталог `webapp/static/images/`.
* Рекомендуемый формат: **JPEG** (по возможности сжатый) или **PNG**. 
* Разрешение: **1920×1080** px (16:9) — хорошо смотрится на большинстве экранов. 
* Размер файла: не более **500 KB** для быстрой загрузки.
